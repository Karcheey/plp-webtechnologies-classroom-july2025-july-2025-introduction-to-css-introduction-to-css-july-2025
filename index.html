What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages — for example, to change fonts, colors, spacing, and positioning of elements.

The Box Model

Every HTML element can be considered a box. The CSS Box Model consists of:

  • Content – text or images inside the element
  • Padding – space between content and border
  • Border – wraps padding and content
  • Margin – outermost spacing between elements
